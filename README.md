# 🚀 Tactic Solutions - Software House Website

A modern and responsive software house website built with React.  
It showcases services, company information, and a contact form with validation.

---

## ✨ Features
- Modern UI/UX design
- Responsive layout (mobile + desktop)
- Home, About, Services, Contact pages
- Contact form with validation (React Hook Form)
- Smooth navigation using React Router

---

## 🛠️ Tech Stack
- React JS
- React Router DOM
- CSS3
- React Hook Form

---

## 📸 Pages
- Home Page (Hero + Services overview)
- About Page (Company info)
- Services Page (What we offer)
- Contact Page (Validated form)

---

```bash
npm install
npm start
